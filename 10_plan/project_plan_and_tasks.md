## 1. 项目概述

### 1.1 项目目标

**总目标**：12周内建成可用的AI内容生产与分发系统，实现从内容采集到多平台分发的自动化闭环。

### 1.2 项目范围

**包含**：
- 极客时间/大家拉/YouTube内容采集
- 双风格内容创作（技术探索者+商业分析师）
- 公众号+小红书自动分发
- 飞书知识库管理
- 数据回流与复盘

**不包含**：
- 视频自动生成（第二阶段）
- B站/抖音自动发布（第二阶段）
- 独立Web界面（依赖飞书）

### 1.3 项目约束

| 约束类型 | 约束说明 |
|---------|---------|
| 时间 | 12周（3个月） |
| 人力 | 1个开发者（你）+ AI协助 |
| 预算 | API成本约$150/月，服务器$30/月 |
| 技术 | 使用Claude Code Skills + n8n + 飞书 |
