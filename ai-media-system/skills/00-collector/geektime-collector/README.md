# Geektime Collector

A Python-based content collector for Geektime (极客时间) articles.

## Installation

```bash
pip install -r requirements.txt
playwright install chromium
```

## Configuration

Create a `.env` file:

```bash
GEEKTIME_COOKIE=your_cookie_here
```

## Development

Run tests:

```bash
pytest tests/
```
